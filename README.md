# ArtJoy - Digital Curation

ArtJoy is a premium digital art gallery and subscription platform that offers curated collections and interactive art generation features. [cite_start]Inspired by modern minimalist design, it provides users with a seamless experience to discover, favorite, and manage digital artwork.

## Features
- **Curated Gallery**: High-quality digital art pieces.
- **Member Profiles**: Save favorites and track your collection.
- **Responsive Design**: Optimized for mobile and desktop viewing.
- **Dynamic Theming**: Built-in support for dark and light modes.
